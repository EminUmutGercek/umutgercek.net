Manual of errno is pretty good. But I’ll summarize it shortly.

errno is a integer global variable declared in <errno.h>.

  • It’s default value is 0.
  • When error happened Library functions or system calls sets it a non-zero value.
  • We can look this value to identify what was the error.

What is system call
