C++ Initializer List

2021-02-01
C++ , Modern C++

What is Initializer List in C++

Initializer list is plain old array C initializations.


 int foo[] = {1,2,3,4,5};

Or STL Container initializations.


 std::vector<int> foo = {1,2,3,4,5};

Create Initializer List Constructor

Need to incldue initializer_list.


 #include <initializer_list>



 class Foo {
   vector<int> m_vec;
 
 public:
   Foo(const initializer_list<int> &v) {
     for (initializer_list<int>::iterator itr = v.begin(); itr != v.end(); itr++)
       m_vec.push_back(*itr);
   }
 }
